# Visual review

Playwright captures the production build at 1440x900, 768x1024, and 375x812. A 720-CSS-pixel viewport also models a 1440px browser at 200% zoom. Tests run in Microsoft Edge.

## Review findings

- Confirmed the editorial layout reads clearly across dark and light sections.
- Replaced the former portrait and project placeholders with an optimized real portrait and three evidence-backed project records.
- Kept the resume action as a functional email request because no verified public resume URL was available.
- Verified the hero, project grid, capability rows, experience, process, certificate carousel, and contact section at every target width.
- Verified no horizontal overflow, console errors, hydration errors, or broken local asset requests.
- Verified keyboard navigation, visible focus, the skip link, 44px controls, mobile-menu dismissal, reduced motion, touch dragging, and a single H1.

## Final result

The visual system is rectangular, typographic, and asymmetric, with restrained blueprint details and blue accents. Project cards use real screenshots and repository links; live-demo actions appear only for verified deployments. No fake metrics, testimonials, skill ratings, or unverified links were added.
